# Jinjei's Dotfiles
---
- `./setup_all.sh` will link dotfiles with `stow`

## How to setup
1. install brew
1. run `brew install stow`
1. install oh-my-zsh
1. in `.zshrc` update your oh-my-zsh file
1. install pyenv by `brew install pyenv`
1. install pyenv virtualenv
1. in `.gitconfig` fix email, name
1. run `./setup_all.sh`
